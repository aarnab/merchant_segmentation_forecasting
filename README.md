# Merchant RFM Segmentation & Volume Forecasting Model

## Project Overview
An end-to-end data analytics pipeline designed to evaluate merchant performance, execute behavioral segmentation, and forecast future payment volumes using statistical regression. 

### Key Features
* **RFM Segmentation (SQL):** Analyzes multi-dimensional transaction datasets to categorize merchants into business tiers (Enterprise, Mid-Market, Churn Risk) based on Recency, Frequency, and Monetary metrics.
* **Predictive Forecasting (Python):** Utilizes `scikit-learn` and linear regression techniques to analyze historical transaction growth and forecast future Gross Merchandise Value (GMV).
* **Business Insights:** Identifies at-risk merchants for retention campaigns and projects future gateway load requirements.

## Tech Stack
* **SQL:** Data Extraction, Aggregation, Rule-Based Segmentation
* **Python (Pandas, Scikit-Learn):** Scripting, Statistical Regression, Predictive Modeling
